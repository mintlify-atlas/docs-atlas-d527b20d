This guide will get you up and running with MobaXterm Discord RPC in just a few minutes.

## Before you start

Make sure you have:
- Windows operating system
- Administrator access
- MobaXterm installed
- Discord installed and running

## Quick installation

<Steps>
  <Step title="Download files">
    Get the latest release from [GitHub](https://github.com/poloputoamo/mobaxterm-discord-rpc/releases). Download:
    - Your preferred version (`mobaxterm-complete` or `mobaxterm-silence`)
    - `discord-rpc.dll`
    - Required DLL file for `final.dll`
  </Step>

  <Step title="Backup and replace">
    1. Navigate to `C:\Program Files (x86)\Mobatek\MobaXterm`
    2. Backup the original `MobaXterm.exe`
    3. Rename your downloaded version to `MobaXterm.exe` and place it in the directory
    4. Add both `discord-rpc.dll` and `final.dll` to the same directory
  </Step>

  <Step title="Launch and verify">
    1. Run MobaXterm as administrator
    2. Open Discord
    3. Check your Discord profile for "Playing MobaXterm"
  </Step>
</Steps>

## Verify it's working

### Check Discord profile

Once MobaXterm is running, your Discord profile should show:

- **mobaxterm-complete**: Displays detailed activity like "Terminal activa" or "Editando archivos" when you're using terminals or the text editor
- **mobaxterm-silence**: Simply shows that MobaXterm is running without detailed status

### Test the integration

If you installed **mobaxterm-complete**, try:

1. **Open a terminal session** - Discord should update to show terminal activity
2. **Open the MobaRTE text editor** - Discord should show you're editing files
3. **Start a local bash session** - Discord should indicate bash activity

Status updates appear within 1-2 seconds of changing activity.

## Common issues

<AccordionGroup>
  <Accordion title="Discord RPC not appearing">
    - Make sure Discord is running **before** you launch MobaXterm
    - Launch MobaXterm with administrator privileges
    - Check Discord settings: User Settings > Activity Privacy > "Display current activity" must be enabled
    - Restart both Discord and MobaXterm
  </Accordion>

  <Accordion title="MobaXterm won't start">
    - Restore your backup `MobaXterm.exe`
    - Verify all three files are in the directory: `MobaXterm.exe`, `discord-rpc.dll`, and `final.dll`
    - Make sure you renamed the files correctly
    - Try downloading the files again
  </Accordion>

  <Accordion title="Status not updating">
    Only applies to `mobaxterm-complete`:
    - The integration updates every 500ms
    - Make sure the processes are actually running (MoTTY.exe for terminals, MobaRTE.exe for editor)
    - Try closing and reopening the terminal or editor
  </Accordion>
</AccordionGroup>
